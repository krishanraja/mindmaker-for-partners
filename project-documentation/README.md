## Quick Start

**For new developers:**
1. Read [PURPOSE.md](./PURPOSE.md) to understand what we're building
2. Review [ICP.md](./ICP.md) to understand who we're building for
3. Study [ARCHITECTURE.md](./ARCHITECTURE.md) for technical overview
4. Follow [REPLICATION_GUIDE.md](./REPLICATION_GUIDE.md) to set up locally

**For design work:**
1. Start with [BRANDING.md](./BRANDING.md)
2. Reference [DESIGN_SYSTEM.md](./DESIGN_SYSTEM.md) for all tokens
3. Check [VISUAL_GUIDELINES.md](./VISUAL_GUIDELINES.md) for constraints

**For debugging:**
1. Check [COMMON_ISSUES.md](./COMMON_ISSUES.md) first
2. Review [DECISIONS_LOG.md](./DECISIONS_LOG.md) for context
3. Consult [HISTORY.md](./HISTORY.md) for past changes
